تیپی وەرزشی ( بروسک ) لە ساڵی {٢٠١٢} لە سەر دەستی { کاروان باپیر ناسراو بە مەلا کاروان } دروست بووە
سەر بە لیژنەی تیپە میلییەکانی پیرەمەگرون و لیژنەی باڵای ئیدارەی ڕاپەڕینە سەرەتا یەک قۆناغی هەبوو
لە پلە یەکی پیرەمەگرون دواتر توانی نازناوی پلە یەک بەرێت و سەر بکەوێت بۆ پلە نایابەکانی پیرەمەگروون
